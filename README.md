## 🚀 Projeto em Destaque: Sistema de Sincronização e Validação de Dados

Uma solução desktop automatizada desenvolvida para otimizar o fluxo de gestão de dados operacionais:
* **Extração em Tempo Real:** Conecta-se a repositórios na nuvem e consome respostas em CSV dinamicamente.
* **Algoritmo de Qualidade de Dados:** Aplica validação matemática real de CPF (dígitos verificadores da Receita Federal) e padronização de telefones.
* **Formatação Condicional Inteligente:** Destaca inconsistências de dados diretamente no Excel utilizando alertas visuais amigáveis.
* **Independência de Ambiente:** Compilado como executável (.exe) autônomo com interface gráfica responsiva para usuários sem conhecimento técnico.
